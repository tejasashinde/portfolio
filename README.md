# 🌐 Ubuntu-Inspired Portfolio Desktop

This repository contains my interactive Ubuntu-inspired personal portfolio website.

## 📌 Project Purpose

This project is a creative personal portfolio and a technical UI experiment. It combines portfolio storytelling with a desktop operating system metaphor to make browsing projects more engaging than a standard resume website.

Instead of a traditional portfolio page, I wanted to create a web experience that feels like using a lightweight desktop operating system. The portfolio is designed as a playful **Ubuntu/GNOME-style desktop in the browser**, where visitors can explore my work through apps, windows, a dock, terminal commands, project pages, and desktop-style interactions.

This portfolio includes an Ubuntu desktop-like interface where you can explore my profile, skills, experience, projects, resume, and contact links through a mix of GUI-style apps and terminal-style commands.

---

## ✨ Features

- Ubuntu-inspired desktop UI
- GNOME-style dock and app launcher
- Firefox-style project browser
- Terminal-style portfolio interaction
- Interactive desktop apps
- Lock screen and boot screen effects
- Responsive browser-based experience
